## CLI app search for Star Wars characters ##
Returns

* count / resultCount

* name

* filmography

* errors



## Install ##
Please run the following in bash at the root folder

```bash
    npm i
```

## Run ##
Please run the following in bash at the root folder

```bash
    npm run start
```
